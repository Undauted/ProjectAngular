# Projekt Angular - Piotr Kacprowicz
### Aby móc uruchomić projekt nalezy posiadać noda w wersji 5.o i wyżej oraz npm w wersji 3.0 i wyżej.
### Po ściagnięciu
- npm install
- npm start
